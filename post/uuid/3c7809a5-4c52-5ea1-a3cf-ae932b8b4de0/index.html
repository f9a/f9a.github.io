Python syntax sugar

In Python ist alles Syntax-Sugar

>>> l = []

>>> l = list()

>>> class AB: name="AB";

>>> AB.name
"AB"

>>> getattr(AB, "name")
"AB"

>>> type(AB).__getattribute__(AB, "name")
"AB"

>>> 2**2
4

>>> int(2).__pow__(2)
4

>>> class A(object):
  def af(self):
    print "FUNKTION a in Klasse A"

>>> a = A()
>>> a.af()
"FUNKTION a in Klasse A"
>>> type(A).__getattribute__(A, "af")(a)
"FUNKTION a in Klasse A"
